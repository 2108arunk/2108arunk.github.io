Software I am currently using

Operating System

I am currently using EndevourOS is an Arch Based Linux distribution, I always prefer using Linux on my personal machines. I also recommend people to make a switch from Ubuntu to Arch based distributions, there is much more offered here in the Arch world.

Desktop Environment

Using bspwm, polybar and sxhkd. Bspwm is a tiling window manager which tiles windows automatically onto the screen. If you are using Linux and haven't checked out any tiling window manager. I highly recommend trying one. It would feel like a new way to interact with your computer. It is far more efficient than a traditional desktop environment, and also looks very cool. :P

Text editor and IDE

For smaller tasks and projects is use Neovim. I use a preconfigured version called Nvchad. I cannot explain in words the usefulness and extensibility of vim, everyone should spend some time learning vim. OSS verison of Vscode is my IDE of choice, but often times when I want to use a GUI text editor I use Geany.
